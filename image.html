1. Dataset Exploration & Preprocessing

📊 Dataset Overview: CIFAR10

The project utilizes the CIFAR10 (Canadian Institute For Advanced Research - 10) dataset, a standard benchmark for image classification tasks.

Data Source: Hugging Face - CIFAR10 Dataset

Problem Type: Image classification

Key Statistics:

Metric Value
Total Samples 60,000 (50,000 train + 10,000 test)
Unique Classes (Genres) 10
Modalities Visual Information (Image)

Data Characteristics:

  • Images have resolution 32 × 32 pixels.
  • Balanced dataset: each class has 6,000 samples.
  • Blue channel dominates (~70 intensity).
